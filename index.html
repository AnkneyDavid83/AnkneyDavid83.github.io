PhotoPhind

PhotoPhind is a photo library I designed that allows a user to search, browse, and download photos taken by individual photographers from a giant collection. I built this using an API provided by Pexels.com for the purpose of practicing Async.

Colors

Colors is a web application that allows the user to generate and modify random color palettes. The user can also save their color palettes and give each one a name which can be retrieved later. Any color can be created using modification sliders that include alterations of hue, brightness, and saturation. The color's hex code can be copied to clipboard as well for easy use when developing. I created the application with Chroma-js.

See the Pen Colors by David Ankney III (@ankneydavid83)
